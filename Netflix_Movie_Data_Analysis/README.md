## Netflix Movie Dataset Analysis – Project Overview

This project provides a detailed analysis of Netflix movie data using Python. It includes data cleaning, exploratory data analysis (EDA), and visualizations to identify meaningful trends in genres, ratings, durations, and release patterns. The project demonstrates core data analysis skills relevant for academic work, interviews, and professional portfolios.

# Project Highlights

* Cleaned and prepared the Netflix movie dataset for analysis

* Analyzed genre distributions and overall content trends

Examined rating patterns and popularity metrics

Explored variations in movie duration across genres

Identified yearly movie release patterns

Created informative visualizations using Matplotlib and Seaborn

Key Insights

Identification of the most and least represented genres

Genres that tend to receive higher average ratings

Trends in annual movie releases over time

Variation in movie duration across different genres

Broader patterns within Netflix’s movie catalog

Tools & Technologies Used

Python (Pandas, NumPy)

Jupyter Notebook

Matplotlib & Seaborn

CSV dataset

Purpose of the Project

This project demonstrates:

Practical data cleaning and preprocessing techniques

A structured and efficient EDA workflow

Strong data visualization capabilities

Ability to answer data-driven, business-style questions

Development of a polished, portfolio-ready data analysis project
